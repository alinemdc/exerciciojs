Mercado Bitcoin

Essa página descreve a API de Dados, uma forma automatizada de obter os dados de negociações do Mercado Bitcoin, em geral de uso de desenvolvedores de software que desejam integrar essas informações a outros sistemas. Por esse motivo, os métodos têm CORS configurado de maneira a permitir chamadas de outros domínios. A documentação abaixo descreve, com exemplos, como utilizar a interface de dados.
